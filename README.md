# Solapur Chaddur - Traditional Weaving Website

A beautiful, responsive HTML/CSS website showcasing the traditional handwoven Solapur Chaddur from Maharashtra, India.

## 📋 Features

- **Home Page**: Hero section with call-to-action and feature highlights
- **About Page**: Information about the heritage, craft, and mission
- **Products Page**: Showcase of various Solapur Chaddur collections with pricing
- **Contact Page**: Contact information and inquiry form
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern Styling**: Professional design with traditional color scheme

## 🎨 Pages

1. **index.html** - Home page with hero section and features
2. **about.html** - About the tradition and craft
3. **products.html** - Product collection and catalog
4. **contact.html** - Contact information and form
5. **styles.css** - Centralized styling for all pages

## 🚀 Deployment

### GitHub Pages

This website is ready to be deployed on GitHub Pages:

1. Go to your repository settings
2. Navigate to **Settings > Pages**
3. Under "Source", select `main` branch
4. Click Save
5. Your site will be published at `https://dashm-sol.github.io/traditional-chaddar`

### Alternative Hosting Options

- Netlify
- Vercel
- Firebase Hosting
- Any static file hosting service

## 📝 Customization

You can easily customize:

- **Colors**: Edit the color codes in `styles.css`
- **Content**: Update text in HTML files
- **Images**: Replace placeholder images with actual product photos
- **Contact Info**: Update contact details in `contact.html`
